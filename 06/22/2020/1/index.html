Master Theorem

Posted on 06-22-2020 In Algorithms

The master theorem concerns recurrence relations of the form:

T(n)=aT(nb)+f(n)where a≥1,b>1 T(n) = aT\left(\frac{n}{b}\right) + f(n) \quad \text{where} \: a\ge1, b>1 T(n)=aT(bn​)+f(n)wherea≥1,b>1

In the application to the analysis of a recursive algorithm, the constants and function take on the following significance:

  • nnn is the size of the problem.
  • aaa is the number of subproblems in the recursion.
  • n/bn/bn/b is the size of each subproblem. (Here it is assumed that all subproblems are essentially the same size.)
  • f(n)f(n)f(n) is the cost of the work done outside the recursive calls, which includes the cost of dividing the problem and the cost of merging the solutions to the subproblems.

It is possible to determine an asymptotic tight bound in these three cases:

Case 1

Generic form

If f(n)=Θ(nc)f(n)=\Theta(n^c)f(n)=Θ(nc) where c<log⁡bac<\log_bac<logb​a (using Big O notation) then:

T(n)=Θ(nlog⁡ba) T(n)=\Theta(n^{\log_ba}) T(n)=Θ(nlogb​a)

Example

T(n)=8T(n2)+1000n2 T(n)=8T\left(\frac{n}{2}\right)+1000n^2 T(n)=8T(2n​)+1000n2

As one can see from the formula above: a=8a=8a=8, b=2b=2b=2, f(n)=1000n2f(n)=1000n^2f(n)=1000n2, so f(n)=Θ(nc)f(n)=\Theta(n^c)f(n)=Θ(nc), where c=2c=2c=2. It follows from the first case of the master theorem that

T(n)=Θ(nlog⁡ba)=Θ(n3). T(n)=\Theta(n^{\log_ba})=\Theta(n^3). T(n)=Θ(nlogb​a)=Θ(n3).

Case 2

Generic form

If it is true, for some constant k≥0k\ge0k≥0, that:

f(n)=Θ(nclog⁡kn)where c=log⁡ba f(n)=\Theta(n^c\log^kn)\quad \text{where} \: c=\log_ba f(n)=Θ(nclogkn)wherec=logb​a

then:

T(n)=Θ(nclog⁡k+1n) T(n)=\Theta(n^c\log^{k+1}n) T(n)=Θ(nclogk+1n)

Example

T(n)=2T(n2)+10n T(n)=2T\left(\frac{n}{2}\right) + 10n T(n)=2T(2n​)+10n

As we can see in the formula above the variables get the following values: a=2a=2a=2, b=2b=2b=2, c=1c=1c=1, f(n)=10nf(n)=10nf(n)=10n, so f(n)=Θ(nclog⁡kn)f(n)=\Theta(n^c\log^kn)f(n)=Θ(nclogkn), where c=1c=1c=1, k=0k=0k=0. Next, we see if we satisfy the case 2 condition: log⁡ba=log⁡22=1\log_ba=\log_22=1logb​a=log2​2=1, and therefore, c=log⁡bac=\log_bac=logb​a. So it follows from the second case of the master theorem:

T(n)=Θ(nlog⁡balog⁡k+1n)=Θ(n1log⁡1n)=Θ(nlog⁡n). T(n)=\Theta(n^{\log_ba}\log^{k+1}n)=\Theta(n^1\log^1n)=\Theta(n\log n). T(n)=Θ(nlogb​alogk+1n)=Θ(n1log1n)=Θ(nlogn).

Case 3

Generic form

If it is true that:

f(n)=Θ(nc)where c>log⁡ba f(n)=\Theta(n^c) \quad \text{where} \: c>\log_ba f(n)=Θ(nc)wherec>logb​a

then:

T(n)=Θ(f(n)) T(n)=\Theta(f(n)) T(n)=Θ(f(n))

Example

T(n)=2T(n2)+n2 T(n)=2T\left(\frac{n}{2}\right)+n^2 T(n)=2T(2n​)+n2

As we can see in the formula above the variables get the following values: a=2a=2a=2, b=2b=2b=2, f(n)=n2f(n)=n^2f(n)=n2, so f(n)=Θ(nc)f(n)=\Theta(n^c)f(n)=Θ(nc), where c=2c=2c=2. Next, we see if we satisfy the case 3 condition: log⁡ba=log⁡22=1\log_ba=\log_22=1logb​a=log2​2=1, and therefore, c>log⁡bac>\log_bac>logb​a. So it follows from the third case of the master theorem:

T(n)=Θ(f(n))=Θ(n2). T(n)=\Theta(f(n))=\Theta(n^2). T(n)=Θ(f(n))=Θ(n2).
